# Shared Expense Manager

A full-stack placement assignment project to manage shared expenses among groups.

## Tech Stack
- Frontend: React (Vite), React Router, Axios
- Backend: Node.js, Express
- Database: MySQL, Sequelize ORM
- Authentication: JWT

## Quick Start

### Backend
\`\`\`bash
cd backend
npm install
# Set up your .env file
npm start # or node server.js
\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
