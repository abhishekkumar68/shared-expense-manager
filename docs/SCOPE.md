# Project Scope: Shared Expense Manager

## In Scope
- User Registration and Login via JWT
- Group Creation and Membership Management (Join/Leave with timestamp tracking)
- Expense CRUD operations within groups
- Settlement CRUD operations
- Balance Summary generation
- Bulk data import via CSV with anomaly detection
- Relational database schema with MySQL

## Out of Scope
- Fancy UI/UX, Dark mode, Animations
- Chatbots, Notifications
- Analytics, Optical Character Recognition (OCR)
- Microservices Architecture
- Redux for State Management
